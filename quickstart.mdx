## Getting started

Welcome to [Octant](https://octant.sh)! Follow the instructions below to learn how to rent a server and start playing your favourite game!

### 1. Choose a game

Head to [octant.sh](https://octant.sh) and select **Deploy a server**. From there, select the game you'd like to play. Currently we support the following games:

<Frame>
    <img src="/images/site-games.png" />
</Frame>

- [Minecraft](https://octant.sh/games/minecraft)

If a game you'd like to play isn't listed here, feel free to [contact us](mailto:support@octant.sh) and we'll add it in for you!

### 2. Checkout

Once you've selected the game, configure the options and check out:
